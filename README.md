Title: Clustering Indian Listed Companies by Sustainability Maturity — A BRSR Based Analysis of the Chemicals Sector
This repository contains the dataset and code used for the dissertation project. Data was collected from publicly available BRSR annual reports filed on BSE and NSE portals for FY 2023-24.
Contents:
- BRSR Master Dataset FY2023-24.xlsx — sustainability disclosure data for 130 Indian chemicals sector companies
- Python clustering code — to be added during analysis phase
